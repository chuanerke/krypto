# Krypto
Cryptocurrency price info bot made with Discord.py and SQLite3. Created for my Intro to Database Systems class final project. 
Built with Python, using discord.py for the bot, SQLite3 for the database, and yfinance for market data. 
Each feature/command set is divided into cogs.
